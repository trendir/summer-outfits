Yes, you can use the same fabric for lining, and it has several benefits. How to Use the Same Fabric for Lining

Using the same fabric for lining is a simple process. Firstly, choose your fabric, and then cut it out using the same pattern as the outer fabric. Once you have cut out the lining fabric, pin it to the outer fabric, right sides facing each other. Sew along the edges, leaving a small gap to turn the fabric inside out. Once you have turned the fabric inside out, sew the gap, and voila! you have a perfectly lined garment.
