How Much Does A Set Designer Make A Year

Written by Daniel Jul 23, 2023 · 3 min read

Have you ever wondered how much a set designer makes in a year? It's a valid question, especially if you are considering pursuing a career in set design. Understanding the average salary for a set designer can help you determine if it is the right career path for you.

The average salary for a set designer varies depending on factors such as location, experience, and the type of production. However, according to the Bureau of Labor Statistics, the median annual wage for set and exhibit designers was $56,010 as of May 2020.

In general, set designers who work in the motion picture and video industry tend to make the highest salaries. The average annual salary for set designers in this industry was $97,370 as of May 2020. On the other hand, set designers working in performing arts companies earn an average of $61,720 per year.

My Personal Experience

As a set designer with five years of experience, I can attest that the salary range for this profession can vary greatly. When I first started out, I was earning around $35,000 per year. However, as I gained more experience and started working on bigger productions, my salary increased. Now, I earn around $70,000 per year.

Factors That Affect Set Designer Salaries

Several factors can affect the salary of a set designer:

Location

As with many professions, the location where a set designer works can greatly impact their salary. Set designers working in larger cities, such as New York or Los Angeles, tend to earn higher salaries due to the higher cost of living in these areas.

Experience

The more experience a set designer has, the higher their salary tends to be. Experienced set designers who have worked on major productions can earn six-figure salaries.

Education and Training

Most set designers have a bachelor's degree in theater or a related field. However, some set designers have degrees in architecture or interior design. Additional training in computer-aided design (CAD) software can also be beneficial for set designers.

Networking

Networking is an important aspect of any career, including set design. Set designers who have strong connections in the industry may be able to secure higher-paying jobs.

Conclusion of How Much Does a Set Designer Make a Year

Overall, the salary range for set designers can vary greatly depending on factors such as location, experience, and industry. However, with the right education and training, as well as a strong network, set designers can earn a comfortable living while pursuing their passion for creating visually stunning productions.

Question and Answer

Q: How much do entry-level set designers make?
A: Entry-level set designers can expect to earn around $35,000 to $40,000 per year.

Q: What is the highest-paid industry for set designers?
A: Set designers working in the motion picture and video industry tend to earn the highest salaries.

Q: Can set designers work on a freelance basis?
A: Yes, many set designers work on a project-by-project basis as freelancers.

Q: Is it possible to earn a six-figure salary as a set designer?
A: Yes, experienced set designers who have worked on major productions can earn six-figure salaries.
